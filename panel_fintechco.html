Security & Compliance

Built for regulated
environments

Data & Infrastructure
🔒 No intermediary servers
Code goes directly to Claude API via AWS Bedrock or GCP Vertex AI. No third-party server sees your payment code. Unlike Cursor or Windsurf, there is no code index stored on intermediary infrastructure.
🚫 Zero training on your data
Enterprise default — not an opt-out, an opt-in that no enterprise customer has activated. Custom data retention policies. Zero-Data-Retention (ZDR) mode available for PHI/sensitive data.
🇺🇸 US-only inference
Data never leaves US infrastructure. AES-256 at rest, TLS 1.2+ in transit. KMS-backed provider-managed keys on AWS and GCP. BYOK support available — contact your account team for enablement.
Certifications
SOC 2
Type II audited
ISO 27001
encryption std
HIPAA
ZDR eligible
Enterprise Governance Controls
🏢 Managed Settings (MDM)
JSON config deployed via Jamf, Kandji, or Intune. Tamper-resistant at OS level. Engineers cannot override. Controls:
· Approved models (Sonnet default, Opus opt-in)
· MCP server allowlist (no shadow connections)
· PCI-scope directory deny rules
· Per-user & per-team spend caps
· Network egress restrictions
👁 Identity & Audit
SAML 2.0 / OIDC SSO (Okta, Azure AD, Ping). SCIM provisioning. Domain capture — company email auto-routes to enterprise workspace. Audit logs exportable to Splunk, Datadog, Elastic. Compliance API for every action.
🛡 Runtime Security
Human-in-the-loop by default — every file change, command, and network request requires approval. Command injection detection. Suspicious bash commands blocked even if previously allowlisted. Fail-closed matching: unrecognized commands default to manual approval.
🤖 Claude Code Security
Reasoning-based vulnerability scanning. Traces data flows across codebase to catch business logic flaws (broken access control, authorization bypasses) that pattern-matching SAST tools miss. Complements SonarQube — doesn't replace it.
Time to secure deployment SSO: 2–4 hrs · MDM settings: 1 day · Full governance: 1 week

The 2026 market has consolidated around three design philosophies. Claude Code is the only one that covers all three.
